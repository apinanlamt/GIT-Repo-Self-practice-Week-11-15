///// Week 10 //////
วันนี้ผมได้ทำโปรเจกต์เว็บ “Quote Manager” ให้สามารถใช้งานได้จริงครบทุกส่วน โดยมีจุดประสงค์เพื่อฝึกใช้ Fetch API ทำงานร่วมกับ JSON Server ในการจัดการข้อมูลแบบ CRUD (Create, Read, Update, Delete)
เริ่มจากการเตรียมโครงสร้างโปรเจกต์ที่มีอยู่แล้ว ทั้งหน้าเว็บ (index.html), การตกแต่ง (style.css), และไฟล์ฐานข้อมูล (data/db.json) จากนั้นผมไปเขียนและปรับปรุงส่วนของ JavaScript ให้สมบูรณ์ โดยเพิ่มโค้ดใน fetchUtils.js และ quoteManagement.js เพื่อให้เรียก API ได้ครบทุกคำสั่ง เช่น การเพิ่ม ลบ และแก้ไขคำคม รวมถึงการโหลดข้อมูลทั้งหมดมาแสดงบนหน้าเว็บ
ต่อมาผมได้ปรับไฟล์ app.js ให้ควบคุมส่วนของการแสดงผลและการทำงานของปุ่มต่าง ๆ เช่น ปุ่ม Add/Edit สำหรับเพิ่มหรือแก้คำคม และปุ่ม Delete สำหรับลบข้อมูลที่ไม่ต้องการ
ช่วงท้ายของวัน ผมติดตั้งและตั้งค่า JSON Server เพื่อใช้เป็น backend จำลองสำหรับเก็บข้อมูลคำคม โดยใช้คำสั่ง npm run backend ให้เซิร์ฟเวอร์ทำงานที่พอร์ต 5001 จากนั้นเปิดอีกหน้าต่างเพื่อรันเว็บฝั่ง frontend ด้วย npm run dev เมื่อเปิดเว็บในเบราว์เซอร์ ทุกอย่างก็ทำงานได้จริง — ข้อมูลโหลดขึ้นมา, เพิ่ม ลบ แก้ไข ได้ครบถ้วน

ประโยชน์ที่จากการฝึก Practice
วันนี้ผมได้เรียนรู้และทำครบทุกขั้นตอน ตั้งแต่การเชื่อมต่อ front-end กับ back-end ไปจนถึงการทดสอบระบบจริง เว็บ Quote Manager จึงเสร็จสมบูรณ์พร้อมใช้งานในวันนี้
